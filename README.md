## Rule
* 주 단위로 책을 읽는다.
* 일주일에 한 번 온라인 미팅으로 책 내용에 대해 토론한다.
* 한 주마다 리딩을 하는 사람을 랜덤으로 정한다.
* 리딩하는 사람은 주도적으로 한 주의 스터디를 진행하고, 그 외의 멤버들은 책을 읽다가 모르는 점, 토론하고 싶은 점, 모두와 함께 이야기하고 싶은 내용 등 기록하고 싶은 내용을 이슈에 추가한다.
* 스터디 시간 전까지 리딩하는 사람은 이슈들에 대해 답변한다.
